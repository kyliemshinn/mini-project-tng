Flexbox Row

Use these properties to create a Flexbox row layout.

. row {
display: flex:
flex-direction: row;
flex-wrap: wrap;
}

Flexbox Column

Use this to create a Flexbox column layout.

.column {
display: flex;
flex-direction: column;
}

CSS Grid Layout

Build a 12-column layout using CSS grid

.grid {
diplay: grid;
width: 100%;
grid-template-columns:
repeat(12. 1fr);
}
